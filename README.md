# arf6-interactome-analysis
Computational Mapping of the ARF6 Interactome: A Graph-Theoretic Analysis of
Cellular Signaling Hubs — Repository Documentation

================================================================================

Abstract

This repository contains the complete research materials for a computational
investigation into the network topology of the ADP-ribosylation factor 6 (ARF6)
interactome. ARF6 is a small GTPase established in the literature as a
multifunctional regulator of membrane trafficking, actin cytoskeletal dynamics,
endocytosis, autophagy, and innate immune signaling. Using protein interaction
data retrieved from the STRING database (v11.5) and graph-theoretic methods
implemented in Python, this project tests the hypothesis that ARF6 occupies a
dominant hub position within its interactome — functioning not merely as a
component of individual pathways but as a critical integration point that
bridges functionally distinct signaling modules. The repository includes a
peer-reviewed literature synthesis (ARF6_Research_Paper_Final_Revised.docx) and
a fully annotated computational notebook (ARF6_Complete_Analysis.ipynb)
containing twelve analytical sections, five centrality metrics, community
detection, degree distribution analysis, robustness simulation, and a
comparative interactome analysis against the closely related GTPase ARF1.

1. Introduction: Scope and Organization of this Repository

1.1 Background

The ARF subfamily of small GTPases has established roles in vesicular transport
and actin organization. Of all ARF proteins, ARF6 has been found to be the
least evolutionarily conserved and is unique in that it is found on the plasma
membrane and on vesicles near the outer boundary of the cell, distinguishing it
from most ARF GTPases that are located within the Golgi apparatus. This
peripheral localization positions ARF6 as a central regulator of how the cell
interacts with its external environment and has motivated substantial recent
interest in its roles across membrane trafficking, cytoskeletal remodeling, and
immunological sensing.

This repository provides both a written synthesis of the ARF6 literature and a
computational analysis designed to test whether the functional breadth attributed
to ARF6 in the literature is reflected in measurable topological dominance within
its protein interaction network.

1.2 Repository Contents

2. Methods: Computational Analysis Pipeline

2.1 Data Acquisition and Network Construction

Protein interaction data was retrieved from the STRING Database (v11.5) via its
REST API, querying the Homo sapiens proteome (NCBI Taxonomy ID: 9606) for the
fifty highest-confidence interaction partners of ARF6. This yielded 574 raw
interaction edges across 51 proteins. Interactions were subsequently filtered to
retain only those meeting a STRING composite confidence score of >= 0.700, the
established threshold for high-confidence interactions, producing a final
network of 51 nodes and 181 edges. Graph construction and all downstream
analyses were performed using the NetworkX library.

2.2 Centrality Analysis and Composite Hub Score

Four complementary centrality metrics were computed to characterize node
importance from distinct mathematical perspectives.

    Degree Centrality quantifies the fraction of possible connections a protein
    holds within the network, serving as a direct measure of connectivity.

    Betweenness Centrality measures the fraction of all shortest paths in the
    network that pass through a given protein, identifying critical relay nodes.

    Closeness Centrality reflects the average shortest-path distance from a
    protein to all others, indicating how efficiently it can propagate signals
    across the network.

    Eigenvector Centrality scores a protein based on the importance of its
    neighbors, applying the same recursive logic underlying the PageRank
    algorithm to identify proteins connected to other well-connected hubs.

All four metrics were normalized to a 0-1 range and averaged into a single
Composite Hub Score, a custom metric designed to provide a more robust and
multidimensional ranking of protein importance than any individual measure alone.

Table 1: Centrality Metrics and Their Biological Interpretation

| Metric              | Mathematical Basis               | Biological Meaning                    |
|---------------------|----------------------------------|---------------------------------------|
| Degree Centrality   | Fraction of possible connections | Participation in multiple pathways    |
| Betweenness         | Fraction of shortest paths       | Critical relay between pathway modules|
| Closeness           | Average shortest-path distance   | Efficiency of signal propagation      |
| Eigenvector         | Recursive neighbor importance    | Influence through well-connected hubs |
| Composite Hub Score | Average of 4 normalized metrics  | Integrated measure of hub dominance   |

2.3 Community Detection

Functional modularity was assessed using the Greedy Modularity Optimization
algorithm, which iteratively merges protein communities to maximize the
modularity coefficient Q — a value approaching 1 indicating strong community
structure beyond that expected by chance. The resulting communities were
evaluated for correspondence with functionally characterized protein complexes
and signaling modules described in the accompanying literature review.

2.4 Evidence Decomposition

For each of ARF6's direct interaction partners, the seven component evidence
sub-scores provided by STRING were extracted and visualized. These channels
include experimental (escore), database-curated (dscore), text-mining (tscore),
co-expression (ascore), phylogenetic co-occurrence (pscore), genomic
neighborhood (nscore), and gene fusion (fscore). This decomposition permits
evaluation of data provenance and identification of interactions supported by
direct laboratory evidence versus those inferred computationally.

2.5 Network Robustness Simulation

An in silico knockout experiment was performed by simulating the sequential
removal of proteins from the network under two conditions: targeted removal
(hubs first, in order of Composite Hub Score) and random removal (randomized
order with fixed seed for reproducibility). After each removal, the size of the
largest connected component was recorded. The comparative degradation curves
reveal the structural essentiality of hub proteins relative to the network
average.

2.6 Comparative Interactome Analysis

To assess the functional specificity of the ARF6 interactome, an equivalent
network was constructed for ARF1 using identical API parameters and confidence
thresholds. Jaccard overlap between the two interactomes was computed, and
unique versus shared interaction partners were catalogued and visualized. Network
topology statistics (density, average clustering coefficient) were compared
across both networks.

3. Key Findings

3.1 ARF6 is the Dominant Hub of Its Interactome

ARF6 achieves a degree centrality of 1.000, indicating high-confidence
interactions with all other proteins in the network, and a betweenness
centrality of approximately 0.690, meaning that ARF6 lies on roughly 69% of all
shortest communication paths. These values, combined with the highest Composite
Hub Score in the network, computationally validate the central thesis of the
accompanying literature review: that ARF6 functions not as a specialist in any
single pathway, but as a master integrator of membrane and cytoskeletal
signaling.

3.2 Secondary Hubs Correspond to Known ARF6 Collaborators

The graph-theoretic analysis independently identifies RAC1, RHOA, EGFR, SRC,
and RAB11A as secondary hubs. This correspondence with the proteins discussed in
the literature as key ARF6 collaborators — RAC1 and RHOA in cytoskeletal
remodeling, RAB11A in endosomal recycling, and receptor tyrosine kinases in
surface signaling — constitutes a computational corroboration of the biological
narrative. The algorithm finds the same organizational logic that the literature
describes.

3.3 Community Structure Reflects Functional Pathway Architecture

Community detection identifies distinct protein clusters corresponding to the
exocyst complex, Rho/Rac cytoskeletal regulators, and ARF-GAP/GEF regulatory
proteins — functional groupings whose separation mirrors the dual membrane
trafficking and cytoskeletal roles attributed to ARF6 throughout the literature.
ARF6 itself is positioned at the boundary of multiple communities, consistent
with its role as an inter-module bridge.

3.4 Targeted Removal of ARF6 Fragments the Network

The robustness simulation demonstrates that removal of ARF6 produces the largest
single-step decrease in largest connected component size of any protein tested.
This structural result is a computational parallel to the severe in vivo
phenotypes associated with ARF6 deficiency, including male sterility resulting
from loss of intercellular bridges in spermatogonia (Wang et al., 2024) and
impaired phagosome maturation in macrophages (Egami et al., 2011).

3.5 ARF6 and ARF1 Interactomes Show Limited Overlap

The comparative analysis reveals moderate Jaccard overlap between the ARF6 and
ARF1 interactomes, with ARF6-unique partners enriched for plasma membrane and
cytoskeletal functions, and ARF1-unique partners enriched for Golgi-associated
transport machinery. This computational divergence mirrors the distinct
subcellular localizations of the two GTPases described in the literature, where
ARF6 operates at the cell surface and ARF1 within the Golgi apparatus.

4. Installation and Reproduction

4.1 System Requirements

Python 3.8 or higher is required. An active internet connection is necessary for
Parts 1 and 10 of the notebook, which make live calls to the STRING API. All
remaining analyses operate on locally downloaded data.

4.2 Dependency Installation

All required libraries and their minimum versions are specified in
requirements.txt. Dependencies may be installed via:

    pip install -r requirements.txt

4.3 Notebook Execution

The Jupyter notebook may be launched from the repository root directory via:

    jupyter notebook ARF6_Complete_Analysis.ipynb

Cells should be executed sequentially. Parts 1 and 10 require network access.
Setting random.seed(42) is included in Part 0 to ensure reproducibility of any
stochastic operations across all subsequent analyses.

4.4 Data Source

All protein interaction data is retrieved from:

    STRING Database v11.5
    https://version-11-5.string-db.org
    Species: Homo sapiens (NCBI Taxonomy ID: 9606)
    High-confidence threshold: composite score >= 0.700
